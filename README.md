# AntiXray

Repository README for the AntiXray Vintage Story server mod.

The detailed project documentation lives in:

```text
AntiXray/README.md
```

Quick start:

```powershell
dotnet test -c Release
```

The release package is generated at:

```text
AntiXray/bin/Release/AntiXray-1.0.0.zip
```

Install that zip into the Vintage Story server `Mods` folder and restart the server.

For full installation, configuration, admin commands, architecture, validation, and known risks, read `AntiXray/README.md`.
